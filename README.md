📌 Descripción del proyecto

Dynamic Survey Builder es un sistema diseñado para la creación y gestión dinámica de encuestas.
El proyecto consiste en el diseño y desarrollo de una base de datos relacional que permita:

Crear encuestas personalizadas.

Administrar preguntas de diferentes tipos.

Registrar respuestas de los usuarios.

Generar resultados y análisis de datos.

El sistema permitirá almacenar y organizar la información de manera estructurada, garantizando integridad, eficiencia y escalabilidad.


---


🎯 Objetivo general

Diseñar e implementar una base de datos relacional que permita la creación, administración y análisis de encuestas dinámicas, asegurando integridad y organización adecuada de la información.


---


📌 Objetivos específicos

Diseñar el modelo entidad-relación del sistema.

Implementar tablas normalizadas (hasta 3FN).

Permitir la creación de distintos tipos de preguntas (opción múltiple, abierta, etc.).

Registrar respuestas de los usuarios.

Desarrollar consultas para análisis de resultados.

Integrar la base de datos con el sistema de aplicación.


---


💡 Justificación

Muchas encuestas se gestionan mediante herramientas limitadas o poco flexibles.
Este sistema permitirá:

Crear encuestas personalizadas fácilmente.

Centralizar y organizar la información.

Obtener resultados rápidos y confiables.

Facilitar la toma de decisiones basada en datos.
